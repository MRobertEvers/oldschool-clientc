========================================
Windows XP Build Scripts - File Summary
========================================

This document lists all files created for Windows XP builds.

========================================
MINGW SHELL LAUNCHERS (Optional)
========================================

1. mingw_shell.bat
   - Full-featured CMD shell with MinGW
   - Shows tool versions and environment info
   - Custom prompt indicating MinGW mode
   - Best for interactive work
   
2. mingw_shell_simple.bat
   - Minimal CMD shell launcher
   - Quick startup, no fancy output
   - Just sets PATH and launches shell

3. mingw_shell.ps1
   - PowerShell version with colored output
   - Custom prompt function
   - Native PowerShell integration
   - Run with: .\mingw_shell.ps1

Note: Shell launchers are optional - the build scripts
      work without them. Use them for manual builds or
      interactive compilation work.

========================================
BUILD SCRIPTS (Run these)
========================================

1. build_winxp.bat
   - Main build script with verbose output
   - Checks toolchain and SDL2
   - Configures CMake with Windows XP flags
   - Builds scene_tile_test.exe and model_viewer.exe
   - Copies required DLLs
   - Shows detailed build progress
   
2. build_winxp_quick.bat
   - Simplified version for quick rebuilds
   - Minimal output, shows only errors
   - Same functionality as full script
   - Use after first successful build

3. setup_sdl2_winxp.bat
   - Interactive SDL2 setup helper
   - Checks for SDL2 installation
   - Provides download instructions
   - Verifies development files and DLLs
   - Run this FIRST if SDL2 not installed

4. test_winxp_build.bat
   - Verifies build outputs
   - Checks for executables
   - Checks for required DLLs
   - Optionally runs scene_tile_test.exe
   - Useful for deployment verification

5. clean_winxp.bat
   - Removes build-winxp directory
   - Use before clean rebuild
   - Simple cleanup utility

========================================
WORKFLOW
========================================

First Time Setup:
  1. Run setup_sdl2_winxp.bat
  2. Follow SDL2 installation instructions
  3. Run setup_sdl2_winxp.bat again to verify
  4. Run build_winxp.bat
  5. Run test_winxp_build.bat

Subsequent Builds:
  1. Make code changes
  2. Run build_winxp_quick.bat
  3. Test

Clean Rebuild:
  1. Run clean_winxp.bat
  2. Run build_winxp.bat

Deploy to Windows XP:
  1. Run test_winxp_build.bat to verify
  2. Copy entire build-winxp\ folder
  3. Ensure cache\ folder is accessible
  4. Run executables

========================================
CUSTOMIZATION
========================================

To modify MinGW path:
  - Edit toolchain path in each .bat file
  - Look for: set "MINGW_ROOT=..."

To change optimization:
  - Edit OPT_FLAGS in build_winxp.bat
  - Options: -O2, -O3, -Os (size)
  - CPU targets: pentium3, pentium4, i686

To change build directory:
  - Edit BUILD_DIR in each script
  - Default: build-winxp

To enable/disable static linking:
  - Edit EXTRA_LINKER_FLAGS in build_winxp.bat
  - Add/remove: -static-libgcc -static-libstdc++

SYSTEM REQUIREMENTS
========================================

Build Machine (Development):
  - Windows 7+ (64-bit recommended)
  - 2GB RAM minimum
  - CMake 3.7+
  - MinGW i686 toolchain
  - SDL2 development files

Target Machine (Windows XP):
  - Windows XP SP3 (32-bit)
  - Pentium 4 or later (SSE2 support)
  - 512MB RAM recommended
  - DirectX 9.0c
  - Cache folder with game data

========================================
TROUBLESHOOTING QUICK REF
========================================

Problem: SDL2 not found
Solution: Run setup_sdl2_winxp.bat

Problem: Entry point not found (XP)
Solution: Verify -D_WIN32_WINNT=0x0501 flag

Problem: Missing DLL on XP
Solution: Run test_winxp_build.bat, copy all DLLs

Problem: Compiler not found
Solution: Check MINGW_ROOT path in scripts

Problem: Build fails
Solution: Run build_winxp.bat for full output

Problem: Poor performance
Solution: See BUILD_WINXP.md optimization section
